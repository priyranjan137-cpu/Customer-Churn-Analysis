# Customer Churn Analysis

## Overview

In this project, I am analyzing customer churn data from a telecom company to understand why customers leave and what factors contribute to churn. This project helps me strengthen my skills in data cleaning, SQL, Python, data visualization, and business analysis.

## Project Objective

The main objective of this project is to apply my data analytics skills to a real-world business problem. I will use Excel for data cleaning and preparation, MySQL for querying and analysis, Python for exploratory data analysis and visualization, and Tableau for building interactive dashboards.

Through this project, I aim to identify customer churn patterns, understand the impact of factors such as contract type, tenure, internet service, and payment methods, and generate insights that can help improve customer retention and support business decision-making.
